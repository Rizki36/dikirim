## Tech Stack

    -   Node
    -   Express
    -   Typescript
    -   Prisma ORM
-   Frontend
    -   Next JS
    -   Typescript

## How To Run

-   create database `learn_dikirim`
-   init yarn `yarn`
-   bootstrap packages `yarn lerna bootstrap`
-   Generate Prisma Client `yarn prisma`
-   Run backend `yarn be`
-   Run frontend `yarn fe`
